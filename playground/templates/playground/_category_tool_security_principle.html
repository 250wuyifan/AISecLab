Tool 功能与主要风险（字节安全团队「防雷手册」）
Tool 功能 主要安全风险
数据分析计算（eval/REPL）RCE：LLM 输出恶意代码 → 服务器被接管
网页总结（请求 URL）SSRF：打内网、云元数据窃取密钥
文档格式转换XXE/任意文件读取、SSRF
数据库连接/操作SQL 注入、JDBC 攻击面、LOAD FILE
文件内容解析（YAML/Excel 等）反序列化 RCE、SSRF、SSTI
OAuth 授权第三方应用1-click 凭证窃取、过度代理
操作浏览器CSRF、SSRF、Chrome N-day RCE
重点关注
  • N-day 漏洞：浏览器/依赖未修补 → Agent 操作浏览器时被利用 RCE。
  • 过度代理：Agent 持权过大（如全读邮箱）→ 一旦被劫持损失巨大。
  • 服务鉴权：Tool 调用第三方服务时身份校验缺失 → 攻击者冒充 Agent 滥用。
